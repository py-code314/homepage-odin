My work

Battleship

Battleship is a two-player strategy game where each player secretly arranges ships on a grid and tries to guess the locations of the opponent's ships by clicking on grid squares. The objective is to sink all of the opponent's ships before your own are destroyed, using strategic targeting and deduction.

Weather

The weather dashboard project is a web application that displays current weather conditions, temperature, and 7-day forecast for a city using live data from Visual Crossing API. It features a user-friendly interface with dynamic updates, allowing users to easily view key weather details at a glance.

Library

The library app allows users to browse, manage, and track reading progress of books. It provides an intuitive interface for searching collections and connecting with an online reading community for personalized book recommendations and social engagement.

Todo List

The tasks app is a user-friendly tool designed to help individuals organize, prioritize, and track personal and professional tasks efficiently. It features a clean visual dashboard, easy task creation, and status tracking with options to filter by due dates or completion.

Restaurant Website

The restaurant website greets visitors with an introductory section and showcases key features such as the bar, reservations, and special events. The menu section presents detailed descriptions of dishes and their ingredients, while the about section shares the restaurant's unique story, culinary ethos, and commitment to authenticity.

Tic Tac Toe

Tic Tac Toe is a two-player game played on a 3x3 grid where players take turns marking empty spaces with either X or O. The objective is to be the first player to get three of their marks in a horizontal, vertical, or diagonal row to win the game. If all spaces are filled without a winner, the game ends in a draw.
